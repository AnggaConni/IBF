# 🌾 Pantau Iklim Desa
### Village Climate Monitoring & Anticipatory Action System

> A lightweight Progressive Web App (PWA) for monitoring village-level climate risks using satellite data and supporting Anticipatory Action (AA) and Forecast-based Financing (FbF).

## Overview

Pantau Iklim Desa is an experimental Decision Support System (DSS) designed to help villages, governments, NGOs, and humanitarian organizations identify climate-related risks before disasters occur.

Unlike conventional weather dashboards, this application focuses on **decision making**, combining:

- Satellite observations
- Climate indicators
- Social vulnerability
- Adaptive capacity
- Anticipatory Action recommendations

The application works entirely in the browser and can be installed as a Progressive Web App (PWA).

---

## Features

- 🛰 Satellite-based climate monitoring
- 🌧 Rainfall anomaly (SPI approximation)
- 🌱 Soil moisture monitoring
- 🌾 Vegetation observation (NDVI assisted)
- 📍 Multi-village monitoring
- 🗺 Interactive GIS map
- 📈 Climate dashboard
- 🚨 Early Warning System
- 🛡 Risk assessment
- 📄 PDF report export
- 💾 Offline-ready PWA
- 🌐 Multi-language support
- 📱 Mobile friendly

---

## Data Sources

The application utilizes publicly available datasets.

- Open-Meteo Forecast API
- Open-Meteo Historical Archive API
- OpenStreetMap
- Leaflet
- Chart.js

---

## Technology Stack

- HTML5
- Vanilla JavaScript
- TailwindCSS
- Leaflet
- Chart.js
- Progressive Web App (PWA)

No backend is required.
